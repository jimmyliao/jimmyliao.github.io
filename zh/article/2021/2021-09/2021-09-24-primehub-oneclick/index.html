    How to connect to remote 1-click PrimeHub instance

    PrimeHub 3.8 開始提供了 1-click to AWS 的快速部署 PrimeHub 方式，點下 1-Click Install on AWS 之後，填入你的 email 並點下 Deploy 之後，就可以先去喝杯咖啡等待郵件通知 PrimeHub on AWS 部署完成。

    不過從系統管理角度，如何透過 kubectl 連到遠端的 1-click PrimeHub 呢？

    很簡單，在收到的 email 裡面有個連結，會先 redirect 到 AWS CloudFormation 畫面。

    裡面有幾個提示，主要把紅色這段複製貼上到 Terminal 即可。(該裝的 AWS cli 跟 credential 什麼的還是要設定一下)

    又可以快快樂樂使用 K9S 啦！

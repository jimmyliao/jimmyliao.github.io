Host Multple Blog on one Github Pages

Posted on 2016-05-06   |   In Hexo , Blog , Markdown

不同Blog想要放在同個Github Page…?

  1. 建立Repo. Ex. https://github.com/jimmyliao/Life.git
  2. 裡面新增一個source branch

  3. 最簡單的方法，新增index.html
    (optional) 或是把原本site generator(例如hexo)搬同樣一份, 請見步驟4

  4. 4.1 clone一份Project repo, 方便起見還是在remote設定gh-pages & source, gh-pages就是最後發布的branch. 請見 Ref #1
    4.2

$ cd hexo

Move to Hexo

2016-05-05
Hexo Blog Markdown

Contents

A new start for blogging…

  1. Follow this
1
2
3
4
5
6
7
8
9

$ sudo npm install --unsafe-perm --verbose -g hexo
$ mkdir hexo
$ cd hexo
$ hexo init
$ npm install hexo --no-optional
$ npm install  

$ hexo g #完整是hexo generate 生成靜態頁面
$ hexo s #完整是hexo server 運行本地服務器

裝git發布插件，運行：

1

$ npm install hexo-deployer-git --save

安裝完成後在Hexo根目錄通過_config.yml對相關參數進行設置，在最後幾行：

1
2
3
4

deploy:
  type: git
  repo: git@github.com/abc/abc.github.io.git #這裏是你剛獲取的地址
branch: master

1

$ vim ~/.ssh/config

1
2
3
4

host abc.github.com
 HostName github.com
 IdentityFile ~/.ssh/github_rsa
 User git

保存設置，撰寫新文章，然後生成頁面，最後發布頁面：

1
2
3

$ hexo new [layout] <title>
$ hexo g
$ hexo d #完整是hexo deploy 發布到遠程服務器

或是生成頁面和發布頁面用一條命令：

1

$ hexo d -g

P.S. have to hexo clean before hexo g

1
2
3

$ hexo clean
$ hexo g
$ hexo d

Using Crane to Build Custom Image part 2: Update Keras version of TF 2.4.1 Notebook

Nov 13, 2021 · 1 min read · PrimeHub Crane  ·
Share on:

中文版

前一篇 已經介紹如何設定好 Crane 並且成功驗證可以客製一個 Docker Image，這一篇會透過一個使用 PrimeHub 的實際案例：如何從原有的 TensorFlow 2.4.1 Notebook Image (裡面包括 Keras 2.3.1)，更新 Keras 版本到較新的版本 2.4.3。

  • 打開 Crane，直接選取 tensorflow-notebook-v2-4-1 這個版本，或是從這個列表選取 Image URL

  • 在 pip 裡面直接填入 keras==2.4.3，點下 Build

  • Build 完之後，直接在 REMOTE REPOSITORIES 點選 ADD to PrimeHub

  • 填入預計在 PrimeHub Image 需要的資訊，點下 Create

  • 切到 PrimeHub User Portal，確認已經新增成功

  • 透過 Start Notebook Server，可以透過 Terminal 確認一下安裝的版本

1$ env | grep JUPYTER_IMAGE_SPEC
2
3$ pip list | grep Keras

Enjoy!
